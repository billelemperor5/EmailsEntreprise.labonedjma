eM Client — Logiciel Desktop

Client de messagerie professionnel pour Windows & macOS
Interface eM Client — Vue Inbox

À propos d'eM Client

eM Client est un logiciel de messagerie moderne et puissant, compatible avec notre serveur adexcloud.dz. Il offre une interface intuitive avec gestion des emails, calendrier, contacts et tâches — tout en un.

Télécharger eM Client

Télécharger pour Windows
Télécharger pour macOS

Configuration d'eM Client

1
Téléchargez et installez eM Client Rendez-vous sur emclient.com et téléchargez la version adaptée à votre système.
2
Ajoutez votre compte email Lancez eM Client, cliquez sur Menu > Comptes > Ajouter un compte et choisissez "Courrier > Autre".
3
Configurez les paramètres du serveur Remplissez les détails du serveur entrant (IMAP) et sortant (SMTP) selon les standards de Labo Nedjma.
Onglet "Général"
  • Authentification : Cochez "Utiliser ces identifiants". Renseignez votre nom d'utilisateur (Email complet) et votre mot de passe fourni.
Onglet: IMAP
  • Hôte : mail.adexcloud.dz
  • Port : 143
  • Sécurité : Forcer l'utilisation de SSL/TLS
  • Sync : "Télécharger automatiquement les messages..."
Onglet: SMTP
  • Hôte : smtp.adexcloud.dz
  • Port : 587
  • Sécurité : Utiliser SSL/TLS si disponible
  • Auth : Cochez "Le serveur requiert une authentification"

Paramètres du serveur de messagerie Labo Nedjma

Guide de configuration

Suivez les paramètres ci-dessous pour configurer votre client de messagerie (Outlook, Thunderbird, Mail mobile, etc.) avec le serveur adexcloud.dz.

Étapes de configuration

1
Ouvrez les paramètres Accédez aux paramètres de compte dans votre client de messagerie (Outlook, Thunderbird, etc.)
2
Saisissez vos identifiants Entrez votre adresse email complète et votre mot de passe professionnel
3
Configurez le serveur IMAP Serveur: mail.adexcloud.dz — Port: 993 — Chiffrement: SSL/TLS
4
Configurez le serveur SMTP Serveur: smtp.adexcloud.dz — Port: 587 — Chiffrement: STARTTLS
5
Testez la connexion Envoyez un email test pour vérifier que la configuration fonctionne correctement
Important

Utilisez toujours votre adresse email complète comme nom d'utilisateur (ex: prenom.nom@labo-nedjma.com). En cas de problème, contactez le support technique.
